## Aquarium Object Detection

- Used Kaggle Dataset: Aquarium Combined with COCO annotations converted to YOLO format
- Used YOLOv8n for object detection
- Classes include: fish, jellyfish, penguin, puffin, shark, starfish, stingray
- Epochs: 30
- Image size: 640
- Batch size: 16
- mAP50:
  - all: 0.595
  - fish: 0.643
  - jellyfish: 0.482
  - penguin: 0.728
  - puffin: 0.681
  - shark: 0.567
  - starfish: 0.532
  - stingray: 0.524
